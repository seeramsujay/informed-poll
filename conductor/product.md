# Product Vision: Informed Poll

## One-Line Description
AI-powered polling platform helping young voters make informed decisions through swippable candidate cards and a RAG-enhanced chat assistant.

## Problem Statement
Traditional election guides are overwhelming, often partisan, and not optimized for modern mobile-first users.

## Solution Approach
- **Gamified Selection**: Use Tinder-style swipe logic to build a personal ballot.
- **Informed Context**: provide "Neural Sync" (RAG) to answer questions about candidates based on verified civic data.
- **Transactional Voting**: Connect to Firestore for secure, non-partisan polling.

## Core Features
- **Swipeable Candidate Cards**: Tactical gesture-driven interface.
- **Interactive Timeline**: Visualizing the voting journey.
- **AI Assistant (VoteIQ)**: RAG-powered chatbot grounded in Google Civic data.
